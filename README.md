## Getting Started

### Pre Reqs:
* node.js
* npm
* git

### Building

clone and install:

    $ git@github.com:aerogear/aerogear-simplepush-node-client.git

    $ cd aerogear-simplepush-node-client

    $ npm install

### Add to a Project

In your project do

    npm install path/to/aerogear-simplepush-node-client
